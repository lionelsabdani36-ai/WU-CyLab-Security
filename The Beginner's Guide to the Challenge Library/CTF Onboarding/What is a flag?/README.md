# What is a flag?

Our challenges are Capture-the-Flag (CTF) challenges. A flag is a text string found in a challenge that proves you solved it. You submit this flag in the challenge interface to get points!

Flags follow a common format so that there is little doubt about when you find one. The common format is this: picoCTF{...}

Almost all flags will look like this: picoCTF{l33tsp34k_phr4s3_1234abcd}, where l33tsp34k is a "language" that substitutes some letters for numbers. Often the phrase given is something related to solving the challenge, but not always.
